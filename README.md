## AI-Powered Identity Document Verification

### Competition

### Three Tasks Implemented

**Task 1 — Face Matching & Verification**
- Siamese Network for facial recognition
- Face verification between identity photos
- TensorFlow | OpenCV | Computer Vision

**Task 2 — Age Estimation**
- Deep learning model for age estimation from photos
- Facial analysis and regression
- TensorFlow | Transfer Learning

**Task 3 — OCR & Document Fraud Detection**
- Automatic text extraction from identity documents
- Fraud detection on identity cards
- Two versions: baseline + improved architecture
- EfficientNet | OCR | Computer Vision
